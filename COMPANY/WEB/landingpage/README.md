# Cleanwave Recycling Nigeria - Corporate Landing Page

A modern, professional corporate landing website for Cleanwave Recycling Nigeria Limited built with React 18, Vite, and Tailwind CSS.

## Features

- **Modern Tech Stack**: React 18, Vite, Tailwind CSS, Framer Motion
- **Responsive Design**: Mobile-first approach with professional styling
- **Animations**: Subtle Framer Motion animations (200-400ms duration, easeOut easing)
- **Brand Colors**: Custom color scheme (deep green primary, moss green secondary, earth brown/gold accent)
- **SEO Friendly**: Semantic HTML with proper meta tags
- **Performance Optimized**: Lazy loading, optimized assets
- **Accessibility**: ARIA labels, keyboard navigation, screen reader support

## Pages

- **Home**: Hero section, process steps, roles explanation, impact metrics
- **Team**: Board members and leadership profiles
- **Partners**: Stakeholder categories (government, businesses, NGOs)
- **Contact**: Contact form with office details and social links

## Components

- Navbar (sticky with blur effect)
- Footer (comprehensive with links)
- Hero section with CTAs
- Process steps visualization
- Roles explanation cards
- Impact counter with animations
- Contact form with validation
- Team member cards
- Partner category sections

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5174`

### Build for Production

Build the application for production:

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

Preview the production build locally:

```bash
npm run preview
```
